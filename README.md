# BruisedArtrash - Portfolio Website

Portfolio web para el artista "BruisedArtrash", desarrollado con React, TypeScript y TailwindCSS.

## Características principales

- Diseño mobile-first y completamente responsive
- Paleta de colores personalizada inspirada en la obra del artista
- Estilo visual expresionista con texturas y efectos pictóricos
- Componentes reutilizables para galerías y exhibición de obras
- Transiciones y animaciones sutiles para mejorar la experiencia de usuario

## Tecnologías utilizadas

- **Vite**: Como bundler y entorno de desarrollo con hot reload
- **React 19**: Para la interfaz de usuario con componentes funcionales y hooks
- **TypeScript**: Para tipado estático y mejor mantenibilidad del código
- **TailwindCSS**: Para estilizado eficiente y consistente

## Estructura del proyecto

```
/src
  /assets        # Imágenes, fuentes y recursos estáticos
  /components    # Componentes reutilizables (ArtCard, HeroSection, etc.)
  /hooks         # Custom hooks de React
  /layouts       # Layouts para envolver las páginas (MainLayout)
  /pages         # Páginas principales (Home, Gallery, Contact)
  App.tsx        # Componente principal
  main.tsx       # Punto de entrada
  index.css      # Estilos globales y configuración de Tailwind
```

## Instalación y Uso

1. Clona el repositorio
2. Instala las dependencias:
   ```
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```
   npm run dev
   ```
4. Para generar la build de producción:
   ```
   npm run build
   ```
